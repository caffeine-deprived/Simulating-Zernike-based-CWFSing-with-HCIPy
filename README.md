# Simulating-Zernike-based-CWFSing-with-HCIPy
This repository is the (future) home of code created for Klaus Subbotina Stephenson &amp; Kathryn Jackson's 2026 SPIE Copenhagen paper "Simulating Zernike-based CWFSing with HCIPy." This work examined optimal "Z" sensing distances for Curvature WaveFront Sensing by exploring reconstructed Zernike coefficients as a function of propagated distance.
