# Simulating-Zernike-based-CWFSing-with-HCIPy
This repository is the (future) home of code created for Klaus Subbotina Stephenson &amp; Kathryn Jackson's 2026 SPIE Copenhagen paper "Simulating Zernike-based CWFSing with HCIPy." This work examined optimal "+/-L" sensing distances for Curvature WaveFront Sensing by exploring reconstructed Zernike coefficients as a function of propagated distance.
The attached code is in two files:
_functions.py_ which holds the laplacian operator
_SPIE_final.py_ which utilizes _functions.py_ to perform the calculations and produce plots found in the associated SPIE proceedings.
